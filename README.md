# PetClinicSysterm
# 项目简介
该项目是一个宠物管理系统，为大二下学期Java Web实训作业；主要是对用户的请求和后台数据的处理，使用了MVC的设计模式。

# 使用技术
* 前端：HTML+CSS

* 后端：JSP+Servlet+JavaBean+MySQL

# 实现功能

* 登录、退出

* 浏览诊所的兽医以及他们的专业特长

* 浏览宠物的主人（即诊所的客户）的相关信息

* 更新宠物的主人的相关信息

* 向系统中增加一个新客户

* 浏览宠物的相关信息

* 更新宠物的相关信息

* 向系统中增加一个新宠物

* 浏览宠物的历史问诊记录

* 向宠物的访问历史记录添加一次问诊
